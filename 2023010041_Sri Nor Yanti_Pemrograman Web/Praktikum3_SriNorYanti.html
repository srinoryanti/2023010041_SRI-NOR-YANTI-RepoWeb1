Selamat Datang di Halaman Praktikum HTML

HTML (Hyper Text Markup Language) adalah sebuah bahasa pemrograman yang digunakan untuk membuat halaman website. HTML menjadi dasar dari sebuah halaman website. sebuah file HTML di di simpan dengan ekstensi .html dan dapat eksekusi atau diakses menggunakan web browser (Google Chrome, Mozilla Firefox, Opera, dan lain-lain).

Tag Dasar HTML yang Digunakan

Pada halaman ini, kita menggunakan tag-tag dasar seperti heading, paragraf, tabel, daftar, gambar, dan formulir.
